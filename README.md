# 打地鼠

我是做打地鼠遊戲

地鼠是維尼，如果你運氣夠好，還可能遇到病毒喔

打到一隻得一分，病毒五分

若打錯洞或是用錯誤的鍵打就結束遊戲

有音效，如果沒有接聲音的話會一直跳警告

# 普通版


要玩的話，請執行Whac_A_Mole.m。並把在資料夾裡面的圖檔和音檔放進matlab資料夾。

用鍵盤或滑鼠左鍵打

關於用什麼鍵盤打，執行遊戲時會顯示一下子

怕閃太快，這裡先解釋一下

 7       8       9
 
 ●       ●       ●
 
 4       5       6

 ●       ●       ●
 
 1       2       3
 
 ●       ●       ●
 
 其實如果你有右邊的九宮格鍵盤就很簡單

不只打錯洞，用錯誤的鍵打也會結束遊戲

地鼠會在那邊等到你打，不用怕打不到
